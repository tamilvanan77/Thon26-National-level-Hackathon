**Project Description**

This project is a Django-based web application that predicts credit risk using climate and financial data. It includes user authentication, loan application handling, and optional machine learning integration (Random Forest).
The system analyzes climate factors such as rainfall and temperature along with financial inputs to classify risk levels.

**Features**

User Login & Registration
Loan Application Form
Climate-Based Risk Prediction
Machine Learning Integration (Random Forest)
Admin Panel
SQLite Database
Graph Visualization (Optional)

** Technologies Used**

Python
Django
SQLite
HTML / CSS / JavaScript
Scikit-learn (ML)
Pandas
NumPy
Matplotlib
